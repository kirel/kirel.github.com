# kirelabs.org

Personal website and portfolio of **Daniel Kirsch**.

This repository contains the static files for [kirelabs.org](https://kirelabs.org), a collection of software projects and experiments.

## Projects Featured

- **Detexify**: LaTeX symbol recognition using machine learning.
- **Wahlprogramm-Matrix**: NLP analysis of political programs for Code for Germany.
- **Fun(ctional) Javascript**: Presentations and experiments with functional programming in JavaScript.
- **Polevault Competition App**: Angular and Firebase-based app for track and field competitions.
- **Leichtathletik Mehrkampfrechner**: Athletics multi-event calculator.

## Technology

The site is a static website. It serves as a landing page for various sub-projects and professional information.
